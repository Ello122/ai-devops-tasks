# Simple Task API

## 📌 Opis projektu

**Simple Task API** to proste REST API do zarządzania zadaniami, stworzone w **Node.js** z użyciem **Express** oraz **MongoDB**.

Projekt umożliwia:
- tworzenie zadań
- odczytywanie listy i pojedynczych zadań
- aktualizację zadań
- usuwanie zadań
- filtrowanie po statusie i priorytecie
- podstawową autoryzację użytkowników
