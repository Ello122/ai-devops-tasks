# Zadanie 11 – Testy jednostkowe w Jest

## 1. Opis zadania

Celem zadania jest napisanie prostego testu jednostkowego dla funkcji `sum(a, b)` oraz uruchomienie go przy użyciu frameworka **Jest**.

Funkcja `sum` dodaje dwie liczby i zwraca wynik:

```javascript
function sum(a, b) {
  return a + b;
}
